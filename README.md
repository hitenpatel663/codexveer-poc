# Simple Landing Page

A modern, responsive landing page built with React, TypeScript, and Vite. This project features a clean and simple design with essential sections for showcasing your business or services.

## Sections

The landing page includes the following sections:

- **Hero** - Eye-catching hero section with a call-to-action
- **About** - Information about your company or project
- **Services** - Showcase of services or offerings
- **Contact** - Contact form and information

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **React Router** - Navigation and routing
- **Material-UI Styles** - Styling solution
